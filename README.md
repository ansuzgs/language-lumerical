# language-lumerical

Syntax highlighting for Lumerical in Atom.

[![GitHub Logo](https://image.flaticon.com/icons/svg/25/25231.svg =100x20)
 GitHub](http://github.com/ansuzgs/language-lumerical)  
[Atom](https://atom.io/packages/language-lumerical)



# Shortcuts

Some shortcuts:

* `for` + <kbd>Tab</kbd>

```lumerical
for(var = start:end) {
  #body
}
```

* `if` + <kbd>Tab</kbd>

```lumerical
if(var operator var) {
  #body
}
```

* `ife` + <kbd>Tab</kbd>

```lumerical
if(var operator var) {
  #body
} else {
  #body
}
```

* `sw` + <kbd>Tab</kbd>

```lumerical
switchtolayout;
```

* `r` + <kbd>Tab</kbd>

```lumerical
run;
```

* `gr` + <kbd>Tab</kbd>

```lumerical
getresult(monitor, var);
```

* `wr` + <kbd>Tab</kbd>

```lumerical
write(filename, var);
```
